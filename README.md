# 🏦 Loan Approval Analysis & Risk Assessment

## 📌 Project Overview

This project focuses on analyzing loan applicant data to identify the key factors that influence loan approval decisions.
The goal is to perform data cleaning, exploratory data analysis (EDA), and derive meaningful insights from the dataset.

---

## 🎯 Objectives

* Clean and preprocess the dataset
* Handle missing values
* Perform exploratory data analysis (EDA)
* Identify patterns affecting loan approval
* Create meaningful visualizations

---

## 🛠️ Technologies Used

* Python 🐍
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Features

The dataset contains the following features:

* Gender
* Married
* Dependents
* Education
* Self_Employed
* ApplicantIncome
* CoapplicantIncome
* LoanAmount
* Loan_Amount_Term
* Credit_History
* Property_Area
* Loan_Status

---

## 🧹 Data Cleaning

* Handled missing values using mode (categorical) and median (numerical)
* Converted categorical values into appropriate formats
* Created a new feature: **Total_Income = ApplicantIncome + CoapplicantIncome**

---

## 📊 Exploratory Data Analysis (EDA)

Key analyses performed:

* Loan Approval Distribution
* Credit History vs Loan Status
* Total Income vs Loan Status
* Loan Amount vs Loan Status
* Property Area vs Loan Status
* Education, Married, Dependents comparisons

---

## 💡 Key Insights

* Credit History is the most important factor in loan approval
* Income alone is not a strong deciding factor
* Higher loan amounts may reduce approval chances
* Property area and demographic features show moderate influence

---

## 📈 Visualization

Used Seaborn and Matplotlib to create:

* Count plots
* Box plots
* Scatter plots
* Heatmaps

---

## 🏁 Conclusion

This project provides a clear understanding of the factors influencing loan approval decisions.
It highlights the importance of credit history and financial stability in the approval process.
